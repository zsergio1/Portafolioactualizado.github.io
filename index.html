FUNDAMENTOS DE LAS TECNOLOGÍAS WEB
¿Qué temas aprendió?:
  • - Protocolos Web y Desarrollo de Software
  • - Interrelación entre Roles Frontend y Backend en el Desarrollo Web Moderno
  • - Herramientas y Tecnologías: Diseño y Desarrollo en la Creación de Experiencias Digitales Impactantes
Procedimiento:

En la primera semana, adquirí conocimientos fundamentales sobre protocolo TCP/IP y HTTP, roles de desarrollador Frontend y Backend, tecnologías clave como HTML, JavaScript y CSS para la interfaz de usuario, así como herramientas como Node.js, TypeScript y frameworks como Angular. También exploré lenguajes de programación backend como PHP, Java, Python y Ruby, gestores de bases de datos, y la importancia del diseño UI/UX con herramientas como Figma y Adobe XD.

Reflexión:

Explorar desde el protocolo TCP/IP hasta el diseño UX/UI, esta semana inicial reveló la complejidad interconectada esencial para forjar experiencias web impactantes.

Reflexión

Semana 1

¿Qué aprendió? ¿Cómo aprendió?

Inicié explorando conceptos clave de tecnologías web, desde protocolos como TCP/IP hasta roles en desarrollo web. Utilicé Visual Studio Code para prácticas y desarrollé comprensión en la interfaz.

Semana 2

¿Qué aprendió? ¿Cómo aprendió?

Avancé construyendo páginas web con HTML y CSS, comprendiendo la Open Web Platform y practicando diseño responsivo. El laboratorio abordó flexbox y diseño fluido.

Semana 3

¿Qué aprendió? ¿Cómo aprendió?

Enfoqué en el desarrollo Frontend con Bootstrap y Tailwind, creando interfaces y explorando tipografías y layouts. La práctica calificada involucró la creación de un sitio web con framework.

Semana 4

¿Qué aprendió? ¿Cómo aprendió?

Profundicé en el desarrollo Frontend con JavaScript, manejando el DOM, animaciones y gráficos en Canvas. Laboratorio centrado en animaciones con JavaScript.

Semana 5

¿Qué aprendió? ¿Cómo aprendió?

Exploré Frameworks JS, comprendí JSX, props y estilos. El laboratorio incluyó pasar datos con props y children. La semana 06 abordó eventos, renderizado condicional, bucles y consumo de APIs.

Semana 6

¿Qué aprendió? ¿Cómo aprendió?

Continué el desarrollo Frontend explorando eventos, renderizado condicional e iterativo, formularios y consumo de APIs. Se aplicaron promesas, async/await y Axios para obtener y mostrar datos de manera eficiente.

Semana 7

¿Qué aprendió? ¿Cómo aprendió?

Me sumergí en el uso de hooks en React, desde useState hasta useMemo. La evaluación 01 evaluó la comprensión y aplicación de estos conceptos.

Semana 9

¿Qué aprendió? ¿Cómo aprendió?

Exploré Next.js y su implementación de CSR y SSR. Laboratorio práctico incluyó un caso de consumo de API con TypeScript.

Semana 10

¿Qué aprendió? ¿Cómo aprendió?

Continué con Next.js, abordando Static Generation e Incremental Static Regeneration. Se aplicaron estos conceptos en un caso práctico de consumo de API.

Semana 11

¿Qué aprendió? ¿Cómo aprendió?

Se exploraron Server Side Props y Next Auth para manejar la lógica del servidor y la autenticación. Laboratorio enfocado en un subsistema de autenticación con Next JS.

Semana 12

¿Qué aprendió? ¿Cómo aprendió?

Exploré la arquitectura del desarrollo Backend, abordando servidores web, el funcionamiento del lado del servidor y el uso de Java y Jakarta. Utilicé herramientas como IntelliJ Idea y Tomcat para desarrollar aplicaciones con Spring Core y Spring Boot

Semana 13

¿Qué aprendió? ¿Cómo aprendió?

Inicié el desarrollo Backend con Spring, comprendiendo HTTP, MVC y arquitectura API Rest. Laboratorio práctico de desarrollo de una API Rest.

Semana 14

¿Qué aprendió? ¿Cómo aprendió?

Durante esta semana, se creó el servidor Eureka, centrándose en el registro y consumo de microservicios. Exploramos en detalle la configuración avanzada en Eureka.

Reflexión General

¿Qué aprendió? ¿Cómo aprendió?

Durante este curso, he adquirido un conocimiento integral sobre el desarrollo web, desde los fundamentos tecnológicos hasta la implementación avanzada de frameworks y herramientas modernas. Aprendí a construir páginas web con HTML y CSS, exploré librerías como Bootstrap y Tailwind, y profundicé en el dinamismo de JavaScript y su aplicación en el Frontend. La comprensión de frameworks como React y Next.js abrió nuevas perspectivas, y el desarrollo Backend con Spring amplió mi habilidad para construir aplicaciones web completas. Mi aprendizaje se basó en una combinación de teoría y práctica. Utilicé herramientas como Visual Studio Code, participé en laboratorios prácticos, y la interacción en tiempo real durante clases sincrónicas enriqueció mi comprensión.
